## 👨‍💻 About Me

I'm **Azzam Alanazi**, an Information Systems student at **Shaqra University** (College of Computing & Information Technology), based in **Saudi Arabia**.

I build things across multiple domains — from FiveM game server resources to full-stack web apps. I have a passion for clean architecture, bilingual (AR/EN) interfaces, and security-conscious development.

- 🎓 **Studying** Information Systems @ Shaqra University
- 🎮 **Building** FiveM QBCore resources in Lua — custom admin systems, events, NUI interfaces
- 🌐 **Developing** full-stack web apps (HTML/CSS/JS, Node.js, React, MySQL)
- 🔐 **Learning** Information Security — cryptography, access control, network defense
- 🤖 **Exploring** AI integrations — Ollama, Groq, Gemini, local LLMs
- 📍 **Location** Riyadh, Saudi Arabia
- 🏠 Currently working from home
